# Third-Party Notices

## Legacy Stage-Building Components

Stage Bridge includes twelve extracted SketchUp 2017 component definitions for stage-building use. They came from the publicly available `Big Prop File.skp` collection supplied by the project owner, who confirmed that the collection is free to distribute specifically for stage building.

The full collection is not distributed. The extension contains only the mapped definitions needed for walls, USPSA targets, no-shoots, poppers, stacked targets, barrels, barrel stacks, and start markers.

Exact source and extracted-file fingerprints are recorded in `status_check_stage_bridge/components/manifest.json`. Original definition names and normalization behavior are documented in `status_check_stage_bridge/components/SOURCE.md`.

The Stage Bridge project license does not reduce any separate permissions granted for these component files.
