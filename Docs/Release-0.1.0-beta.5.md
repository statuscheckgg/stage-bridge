# Stage Bridge 0.1.0-beta.5

Beta 5 is an unsigned import-and-edit preview for SketchUp Make 2017. It improves the visible Practisim-to-Big-Prop coordination and adds a layman installation guide.

## What Changed

- Removes SketchUp's stock template human during every STG import.
- Maps `uspsa-popper` to the selected leftmost full-size Big Prop popper (`Group#180`).
- Maps white-center vertical target stacks to the selected Big Prop assembly (`Group#275`).
- Builds black hard-cover stacks from a private copy of that assembly with only the white center changed to black.
- Keeps the common plain wall mappings introduced in beta 4.
- Keeps **Validate Stage** and **Export Practisim Stage** visibly disabled.
- Adds an illustrated [Install and Test Guide](https://github.com/statuscheckgg/stage-bridge/blob/main/Docs/Install-And-Test.md).

## Requirements

- Windows x64.
- SketchUp Make 2017.2.2555.
- An existing Practisim `.STG` file.

## Install

Download the RBZ below, then follow the illustrated guide. This prerelease is unsigned; only approve it when downloaded from this official repository.

## Known Limits

- This beta imports and edits for visual testing; it does not export a Practisim-readable STG.
- Unsupported props remain tagged magenta placeholders.
- SketchUp Web, current SketchUp releases, macOS, and iPad are not supported.
- SketchUp Make 2017 is no longer supported by Trimble.
