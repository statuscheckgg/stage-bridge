# Stage Bridge 0.1.0-beta.7

Beta 7 is an unsigned import-and-edit preview for SketchUp Make 2017. It makes the Stage Bridge toolbar understandable at a glance while retaining all beta 6 prop mappings.

## What Changed

- Replaces the repeated toolbar image with distinct icons for **Import Practisim Stage**, **Add Practisim Prop**, **Validate Stage**, and **Export Practisim Stage**.
- Adds a matching information-document icon for **Stage Metadata**.
- Keeps **Validate Stage** and **Export Practisim Stage** visibly disabled, with recognizable grayed-out silhouettes.
- Retains the beta 6 popper, swinger, stacked-target, short-target, and two-foot fault-line mappings without changing stage transforms.

## Requirements

- Windows x64.
- SketchUp Make 2017.2.2555.
- An existing Practisim `.STG` file.

## Install

Download the RBZ below, then follow the illustrated [Install and Test Guide](https://github.com/statuscheckgg/stage-bridge/blob/main/Docs/Install-And-Test.md). This prerelease is unsigned; only approve it when downloaded from this official repository.

After upgrading, confirm Extension Manager reports version `0.1.0-beta.7`. The toolbar should show four different Stage Bridge icons.

## Known Limits

- This beta imports and edits for visual testing; it does not export a Practisim-readable STG.
- Unsupported props remain tagged magenta placeholders.
- SketchUp Web, current SketchUp releases, macOS, and iPad are not supported.
- SketchUp Make 2017 is no longer supported by Trimble.
